সচরাচর জিজ্ঞাসা

API Key কোথায় পাব?▼
console.anthropic.com এ গিয়ে ফ্রি একাউন্ট খুলুন। প্রথমে কিছু ফ্রি ক্রেডিট পাবেন। Settings → API Key → Create Key করুন। তারপর আমাদের টুলের Settings এ পেস্ট করুন।
ফ্রি প্ল্যানে কতগুলো জেনারেশন পাব?▼
ফ্রি প্ল্যানে প্রতিদিন ৩টি AI জেনারেশন পাবেন। প্রতি রাত ১২টার পর রিসেট হবে। Pro প্ল্যান নিলে আনলিমিটেড।
Pro প্ল্যান কিভাবে কিনব?▼
হেল্প ফর্মে "Pro আপগ্রেড" সিলেক্ট করে রিকোয়েস্ট পাঠান। আমরা বিকাশ/নগদ/কার্ড পেমেন্ট লিংক পাঠাব। পেমেন্টের পর ২৪ ঘণ্টার মধ্যে আপনার অ্যাকাউন্ট আপগ্রেড হবে।
আমার ডেটা কি নিরাপদ?▼
হ্যাঁ। আপনার API Key শুধু আপনার ব্রাউজারে সেভ থাকে, আমাদের সার্ভারে নয়। কন্টেন্ট জেনারেশন সরাসরি Anthropic API-এ যায়।
এজেন্সি সার্ভিসে কতদিনে ফলাফল পাব?▼
সার্ভিসের ধরন অনুযায়ী ভিন্ন। Content Strategy তে ২-৪ সপ্তাহে উন্নতি দেখা যায়। Creator to Business সেটআপ ২-৩ মাসে প্রথম আয় শুরু হয়। গড়ে আমাদের ক্লায়েন্টরা ৩ মাসে উল্লেখযোগ্য পরিবর্তন দেখতে পান।
